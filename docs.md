# UI setup

## Install Vue CLI
https://cli.vuejs.org/

`npm install -g @vue/cli`

`npm i -g @vue/cli-service-global` required for _vue serve_

## Vue serve
`vue serve` 

in the directory with the App.vue file

`vue serve -p 8888 -o` 

serve on port 8888, open browser
